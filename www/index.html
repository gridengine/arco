Accounting and Reporting Console (ARCo)

The optional ARCo add on module enables you to gather live accounting and reporting data from a Grid Engine system and store the data in a standard SQL database. ARCo also provides a web-based tool for generating information queries on that database and for retrieving the results in tabular or graphical form. ARCo enables you to store queries for later use, to run predefined queries, and to run queries in batch mode, for example, overnight.
