Build ARCo

For building ARCo some additional third party tools are needed. The easiest way to get access to these tools is to download the courtesy binaries from the project download list page at:

http://gridengine.sunsource.net/servlets/ProjectDownloadList

Sun Java Web Console (SJWC) 3.0.2, neccessary for installing ARCo, can be obtained:

  • from the SGE binaries
  • installed automatically with Solaris 10 Update 3 or later
  • can be downloaded from here

The following jar files, neccessary for building ARCo, can be obtained from the directories specified below, after the (SJWC) is installed. Location of these file is platform specific.

Next, checkout the source code. Refer to information at:

http://gridengine.sunsource.net/servlets/ProjectSource
 
         % cvs co arco

Create build_private.properties

         % cd checkout_dir/arco/source

         % cp build.properties build_private.properties

Edit build_private.properties

  • specify paths to all the jars
  • specify sge.root
  • specify sge.srcdir

Edit build.site

         ANT_HOME=<somewhere>/apache-ant-1.6.5

         WSDP_HOME=<somewhere>/jwsdp-1.6

         JAVA_HOME=/usr/java/jdk1.5.0_06

Compile

         % build.sh clean

         % build.sh dist

Install ARCo. See Installing ARCo
