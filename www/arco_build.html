Build ARCo

For building ARCo some additional third party tools are needed.

The following libraries can be downloaded from the locations specified below.

Library

Location

jh.home

JavaHelp

jaxb.home

Jaxb

emma.dir

EMMA free Java code coverage tool

postgres.jdbcdriver

This is usually part of the PostgreSQL distribution or can be downloaded at PostgreSQL

oracle.jdbcdriver

This is usually part of the Oracle distribution  [OraHome]/jdbc/lib/classes12.jar

mysql.jdbcdriver

This is usually part of the MySQL distribution or can be downloaded at MySQL

Next, checkout the source code.

 
         % git clone git://github.com/gridengine/arco.git

Create build_private.properties

         % cd arco/source

         % cp build.properties build_private.properties

Edit build_private.properties

  • specify paths to all the jars

  • specify sge.root

  • specify sge.srcdir

Edit build.private

         ANT_HOME=<path to ant installation directory>

Compile

         % build.sh clean

         % build.sh dist

Install ARCo. See Installing ARCo
