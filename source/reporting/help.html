Help for the SUN Grid Engine Accounting & Reporting Console

01.12.2003
  • 1.0 I18N
    For Mozilla choose Edit -> Preferences -> Languages. Currently supported languages are :
    • de
    • de_DE
    • de_DE_EURO
    • en_US
    • fr_FR
    • Default
      (contains only keys)
    • Only a few jsps are currently internationalised (QueryList). Messages, coming from the MODEL are not (yet) internationalised (BreadCrumb, ...).

  • 2.0 Configure View
    The following types are supported:
    • BarChart
    • PieChart
    • LineChart
    • Stacked LineChart
    • Tabular
    • Pivot
